# Job Application Tracker

Application to track job applications and their statuses

## Use Cases

### Track Job Applications
Record and categorize job applications, their statuses, and relevant details

### Manage Job Categories
Organize job applications into categories based on job type or source

### Monitor Application Statuses
Update and track the status of each job application


## Data Model

### job_applications
Stores job application records

### categories
Stores job application categories

### statuses
Stores application statuses
